ESTRATÉGIAS PARA PROMOVER UMA GESTÃO PARTICIPATIVA – PARTE 2

Para que o funcionamento dos colegiados gestores alcance os objetivos visados, é preciso atentar para o seguinte conjunto de questões operacionais:

Os colegiados gestores são compostos por representantes indicados de todos os setores e grupos de trabalho do serviço.

As equipes de trabalho devem ter permissão para fazer discussões entre seus membros antes das reuniões.

Reuniões precisam acontecer com regularidade.

Reuniões com pautas definidas, para que a mobilização em torno dele se mantenha.
